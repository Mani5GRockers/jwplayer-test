Quality toggle should work with multiple streaming sources using Caterpillar, Shaka or Flash-providers, like it does when using HTML5-provider.

Configuration reference:

https://developer.jwplayer.com/jw-player/docs/developer-guide/customization/configuration-reference/#quality-toggle-for-video-files

Sources:

Multiple sources of the same type using a streaming protocol (HLS or DASH) with different non-adaptive qualities.

Expected behavior

Both sources are available and the quality toggle is displayed, like it is when using HTML5-provider.

Actual behavior

Only the first source plays and quality toggle is not available when using Caterpillar, Shaka or Flash-providers.

Settings:

jwplayer('player').setup({
    "title": "One playlist item with multiple quality sources using streaming",
    "hlshtml": true,
    "sources": [{
           "file": "https://streaming4.ur.se/urse/_definst_/mp4:196000-196999/196679-9.mp4/playlist.m3u8",
           "label": "SD"
        },
        {
            "file": "https://streaming4.ur.se/urse/_definst_/mp4:196000-196999/196679-6.mp4/playlist.m3u8",
            "label": "HD"
        }
    ]
});

JW Player version

7.7.1 with Enterprise key

Stream server

Wowza Streaming Engine 4 Perpetual Edition 4.5.0.01 build18956
